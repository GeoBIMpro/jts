The JTS Topology Suite is an API of spatial predicates and functions for processing geometry. Version history.

JTS WebApp

JTS WebApp is a testing ground for bringing new (and old) JTS features to the browser. For more details see the source of the project.

  • ol3 with buffer of geometry and WKT / WKB tests
  • fetching, parsing, execution of sample xml test
  • simple test of geojson reader / writer
